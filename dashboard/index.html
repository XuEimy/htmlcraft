反 AI 通用化 — Slop 指纹检查
绝对禁止 ✗
✗ 紫/青色渐变光斑背景
✗ Glassmorphism 毛玻璃卡片叠加
✗ Hero 区四格等宽「亮点」并排
✗ 过度使用渐变文字
✗ 漂浮的 3D 几何装饰元素
✗ 默认 Inter/Roboto，无字体个性
✗ 空泛文案「The future of X is here」
主动寻找 ✓
✓ 一个主导视觉锚点
✓ 字号对比 ≥ 3× (最大 vs 最小)
✓ 真实的排版个性
✓ 不对称或显式网格纪律
✓ 区域密疏节奏变化
✓ 参考图深度分析（2 轮）
✓ 视觉论点驱动每个决策
